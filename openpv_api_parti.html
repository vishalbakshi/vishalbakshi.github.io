ObjectiveÂ¶

The National Renewable Energy Laboratory (NREL) offers access to a variety of APIs related to clean energy data across the United States. The OpenPV project provides "real-time status of the solar photovoltaic market in the U.S.". In this post, I explore the use of the OpenPV API, mainly to practice importing and converting JSON files into a useful pandas DataFrame, and to learn how to use the Basemap package for visualization.

When I looped through each state for the state parameter, I kept retrieving only 49/50 states. California's data was not being returned so I create a seperate request for state=CA (there were a total of 628616 installs in for California in the NREL database so I chose 10000 installs per page over 63 pages):

        ca = json.loads(a.get(url).text)
        cas.append(ca)
    except:
        continue

Each JSON output includes a list of results dictionaries, and each dictionary contains the following outputs:

  • ID
  • Cost per Watt
  • Data Installed
  • Size (in kW)
  • State
  • Zipcode

I created DataFrames for the 49 states and California using json_normalize, appended the CA DataFrame to the DataFrame with the other 49 states' install info, and exported them to a pickle file.

Visualizing the Data: Bar PlotÂ¶

I want to visualize the average cost per Watt for each state, so I drop NaN and 'n/a' records from the cost and size_kw fields, and then create a bar plot showing the average cost per watt for each state:

In [3]:
cost_df = data.dropna(subset=['cost', 'size_kw'])
cost_df = cost_df.query('cost != "n/a"')
In [16]:
costs = [group[1].cost.astype(int).mean() for group in cost_df.groupby('state')]
kws = [group[1].size_kw.astype(int).mean() for group in cost_df.groupby('state')]
avg_cost = [cost / (1000*kw) for cost, kw in zip(costs, kws)]

Visualizing the Data: BasemapÂ¶

A better visualization of the average cost can be done on a map of the United States using a colormap corresponding to the cost range. I referenced fillstates.py on GitHub as well as a relevant stackoverflow post to create this map. I start by converting the avg_cost list to a dictionary with the full state names as keys, and then plot the map using the aforementioned references:

In [11]:
avg_cost = dict(zip(state_names,avg_cost))
In [14]:
fig, ax = plt.subplots(1)
fig.set_size_inches(15, 10)

m = Basemap(llcrnrlon=-130, llcrnrlat=18, urcrnrlon=-60,urcrnrlat=54.9, projection='lcc', lat_1 = 33, lat_2=45, lon_0=-95)
shp_info = m.readshapefile('cb_2016_us_state_500k', 'states', drawbounds=True)
cost_dict = dict(zip(states, avg_cost))
colors = {}
statenames = []
cmap = plt.cm.Oranges
vmin = min(avg_cost.values())
vmax = max(avg_cost.values())

for shapedict in m.states_info:
    statename = shapedict['NAME']
    if statename in state_names:
        cost = avg_cost[statename]
        colors[statename] = cmap(1.-np.sqrt((cost - vmin)/(vmax-vmin)))[:3]
    statenames.append(statename)

for nshape, seg in enumerate(m.states):
    if statenames[nshape] in state_names:
        if statenames[nshape] == 'Alaska':
            seg = list(map(lambda (x,y): (0.35*x + 750000, 0.35*y + 1250000), seg))
        if statenames[nshape] == 'Hawaii':
            seg = list(map(lambda (x,y): (1.3*x + 3800000, 1.3*y-1600000), seg))
        
        color = rgb2hex(colors[statenames[nshape]])
        poly = Polygon(seg, facecolor=color, edgecolor=color)
        ax.add_patch(poly)
        

norm = Normalize(vmin=vmin, vmax=vmax)
mapper = ScalarMappable(norm=norm, cmap=cmap)
cax = fig.add_axes([0.27, 0.05, 0.5, 0.05])
cb = ColorbarBase(cax, cmap=cmap, norm=norm, orientation='horizontal')
cb.ax.set_xlabel('Average Cost per Watt (USD)')
ax.set_title('Cost per Watt in Dollars for PV Installs in U.S. States', fontsize=16)
plt.show()

The average cost per watt corresponds to the lightness of the color orange, so states like Alaska and Montana have a lower cost per watt than say Iowa or Vermont.
