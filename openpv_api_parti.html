ObjectiveÂ¶

The National Renewable Energy Laboratory (NREL) offers access to a variety of APIs related to clean energy data across the United States. The OpenPV project provides "real-time status of the solar photovoltaic market in the U.S.". In this post, I explore the use of the OpenPV API, mainly to practice importing and converting JSON files into a useful pandas DataFrame, and to learn how to use the Basemap package for visualization. This process also leads me to some comparisons between average cost per Watt for different states and install sizes across the U.S.

Testing for Statistical SignificanceÂ¶

I want to check the p-values of the difference in means between the size ranges. In order to determine if I can use the ttest_ind function from the scipy package, I'll check to see if the Central Limit Theorem applies to the distribution of avg_cost for each size range:

In [164]:
from scipy import stats

for i, df in enumerate(sizes):
    fig, ax = plt.subplots(1)
    x = [np.sum(np.random.choice(df.avg_cost.values, size=100, replace=True)) for _ in range(1000)]
    res = stats.probplot(x, plot=ax)
    ax.set_title(labels[i], fontsize=16)
    plt.show()

For each size range, the distribution of the sums of sample values chosen at random are (very) close to being normally distributed so the CLT holds for the distribution of avg_cost.

First I check the p-value of each individual sample mean using ttest_1samp where the null hypothesis states that the mean of the sample is equal to the mean of the population. In this case the calculated mean of all values in a given size range are the "population":

In [221]:
print('Size Range\tCalculated Mean\tp-value')
for i, df in enumerate(sizes):
    np.random.seed(1)
    mu = df.avg_cost.values.mean()
    p_value = stats.ttest_1samp(np.random.choice(df.avg_cost.values, size=int(0.1*len(df.avg_cost)), replace=True), mu)[1]
    print(labels[i]+'\t%0.3f\t%0.3f' % (mu, p_value))

10000 to 500000 Watts	5.373	0.222

For each Size Range the null hypothesis holds and there is strong to very strong evidence that the sample mean is equal to the population mean. Next, I compare the means from different size ranges and test the null hypothesis that the mean of each sample is identical. Since the length of each array must be equal, I chose the sample size to be 1000 which is about half of the smallest set of values, 2215 for sizeC:

In [231]:
len(sizeA.avg_cost), len(sizeB.avg_cost), len(sizeC.avg_cost)

print('Means\tp-value')
perms = [(0, 1), (1, 2), (0, 2)]
labels = ['A vs B', 'B vs C', 'A vs C']

for i, perm in enumerate(perms):
    np.random.seed(2)
    sample1 = np.random.choice(sizes[perm[0]].avg_cost, 1000, replace=True)
    sample2 = np.random.choice(sizes[perm[1]].avg_cost, 1000, replace=True)
    p_value = stats.ttest_ind(sample1, sample2, equal_var=False)[1]
    print(labels[i] + '\t%0.5f' % p_value)

ttest_ind is a two-sided test, so a one-sided test would have half the resulting p_values. From this test there is strong evidence to reject the null hypothesis and I can make a significant claim, based on this model, that as the size of the PV system increases the average cost decreases as demonstrated in Table 1 above.

Caveats and Next StepsÂ¶

The issue of selecting sample size is the first source of my hesitancy to make a strong claim from this analysis, since I arbitrarily chose 1000 samples to compare the means of avg_cost from each size range. Next, collecting the mean avg_cost of each size range across different states into one array and concluding that the difference in means is significant overlooks the possibility that different states may use a different set of practices of designing, constructing and managing different sized PV systems, something that may affect the avg_cost of each type of system.

In order to further analyze these assumptions and potential sources of errors, I'll take a look in a future post at other sources for PV install data, such OpenPV's CSV dataset, which may contain more features of interest regarding the installs, clarifying the difference between the installs across different states.
