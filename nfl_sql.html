ObjectiveÂ¶

Carnegie Mellon University's Statistics department, led by Maksim Horowitz, released the <a href = "https://github.com/maksimhorowitz/nflscrapR">nflscrapR</a> R package with which this data was obtained and posted on Kaggle as a CSV file. I explore this data in this post and my focus is to gain some more fluency with SQL queries.

Creating the Table SchemaÂ¶

I started by importing the CSV into a pandas DataFrame using pd.read_csv() and specified the dtype as either numpy.int (8 or 32), numpy.float16, or str for each column. The original CSV contained NA and empty string values which I replaced with '\N', the NULL value equivalent for SQL, using df.fillna('NA', '\N') and df.replace('','\N'). I created a nfl_table.sql batch file to create the table in MySQL, which can be viewed here. Additionally, I modified the header in the CSV to remove the periods (.) from column names (such as Yards.Gained) since this may conflict with SQL's database.table or table.column queries. I also added an Idx header for the first column, which was originally blank, so that I could use it as the table's primary key.

Once created, I validated the MySQL table plays with the pandas Dataframe using Dataframe.column.value_counts.

Initial Data SummariesÂ¶

I start out by asking some summarizing questions:

  • In the 2015 Season, what was the average:
    • YardsGained for Complete passes versus rush attempts?
    • FieldGoalDistance for Good, No Good and Blocked field goals?
    • YardsGained for Short versus Deep passes?

  • In the 2015 Season, what were the percentages of different:
    • RunLocation values: left, middle and right?
    • PassOutcome values: Complete versus Incomplete Pass?
    • PassLength values: Short versus Deep?

Philadelphia Eagles Play-By-Play AnalysisÂ¶

I want to explore the trends present in my beloved Eagles' plays during the 2015 season. The roster was significantly different than the current season, with different starters at critical offensive and defensive positions as well a different head coach so the results will not translate to any insights for the 2017 season. Initially, I view the summary of play counts by PlayType for all recorded Eagles' possessions:

-- Query 7: Summary of PlayType for Philadelphia Eagles' possessions
SELECT  p.PlayType, 
        COUNT(p.PlayType),
        p.posteam
FROM plays p
WHERE p.posteam = 'PHI'
GROUP BY p.PlayType
ORDER BY COUNT(p.PlayType) DESC;
Figure 7: Query 7 Results

Next, I extract the top 10 receivers (based on # of passes caught) and their average YardsGained per complete reception, and total YardsGained across all recorded receptions:

-- Query 8: Total YardsGained, Average YardsGained for top 10 receivers (based on Number of Passes caught)
SELECT  p.PlayType,
        p.PassOutcome,
        p.Receiver,
        AVG(p.YardsGained),
        SUM(p.YardsGained), COUNT(*) AS PassesCaught 
FROM plays p
WHERE p.PassOutcome = 'Complete' AND p.PlayType = 'Pass' AND p.posteam ='PHI'
GROUP BY p.Receiver
ORDER BY COUNT(*) DESC, SUM(p.YardsGained) DESC, AVG(p.YardsGained) DESC
LIMIT 10;
Figure 8: Query 8 Results

I then extract the top rushers (based on number of rushing attempts) and average YardsGained for each RunLocation of left, middle and right.

-- Query 9: Top rushers (based on number of rushing attempts) and their average YardsGained for each RunLocation
SELECT  p.posteam,
        p.PlayType,
        p.Rusher,
        COUNT(*) AS RushAttempts,
        p.RunLocation,
        AVG(p.YardsGained)
FROM plays p
WHERE p.PlayType = 'Run' AND p.posteam = 'PHI' AND p.Rusher IS NOT NULL AND p.RunLocation IS NOT NULL
GROUP BY p.Rusher, p.RunLocation
ORDER BY p.Rusher, COUNT(*) DESC, AVG(p.YardsGained) DESC
LIMIT 10;
Figure 9: Query 9 Results

The starting running back (Demarco Murray) had the highest number of RushAttempts overall, with the highest average YardsGained when he rushed to the middle. Interestingly enough, he had the fewest attempts in that direction. The second most used running back (Darren Sproles) was more successful when he rushed right, which was also the direction in which he ran the most.

A more interesting analysis is to identify the most successful rushing attempts based on how well the opposing team defended the run. I'll measure a team's effectiveness at defending the run based on the average YardsGained per rush attempt against their defense. First I extract the top 10 teams with the lowest average rushing YardsGained allowed all season:

-- Query 10: Top 10 teams with the lowest average rushing YardsGained allowed all season
SELECT  p.DefensiveTeam,
        p.PlayType,
        AVG(p.YardsGained) AS Yards_Per_Rush_Attempt
FROM plays p
WHERE p.PlayType = 'Run' AND p.DefensiveTeam IS NOT NULL
GROUP BY p.DefensiveTeam
ORDER BY AVG(p.YardsGained)
LIMIT 10;
Figure 10: Query 10 Results

Next I'll compare Philadelphia's success on rush attempts versus those top 10 teams:

-- Query 11: Eagles' average YardsGained per rush attempt against top 10 defenses (based on average rushing YardsGained allowed per game over season)
SELECT  p2.posteam AS Offense,
        TopDefRush.DefensiveTeam AS Defense,
        p2.PlayType,
        AVG(p2.YardsGained) AS Philly_AvgRushYds,
        TopDefRush.Yards_Per_Rush_Attempt AS AllTeams_AvgRushYds,
        AVG(p2.YardsGained) - TopDefRush.Yards_Per_Rush_Attempt AS AvgRushYds_Diff
FROM (
    SELECT  p1.DefensiveTeam,
            p1.PlayType,
            AVG(p1.YardsGained) AS Yards_Per_Rush_Attempt
    FROM plays p1
    WHERE p1.PlayType = 'Run' AND p1.DefensiveTeam IS NOT NULL
    GROUP BY p1.DefensiveTeam
    ORDER BY AVG(p1.YardsGained) LIMIT 0,10) AS TopDefRush
INNER JOIN plays p2
 ON p2.DefensiveTeam = TopDefRush.DefensiveTeam
WHERE p2.posteam = 'PHI' AND p2.PlayType = 'Run'
GROUP BY TopDefRush.DefensiveTeam
ORDER BY AVG(p2.YardsGained) DESC;
Figure 11: Query 11 Results

The Eagles faced 5 of the top 10 defenses (based on average rushing yards allowed per rush attempt), rushed more yards than average against the Carolina Panthers and the Tampa Bay Buccaneers and rushed fewer yards than average against the New England Patriots, Arizona Cardinals and New York Jets.

Based on this, it seems that the opposing defense's effectiveness against the run does not seem to dictate how well the Eagles run the ball. To further explore this point, I'll compare Philadelphia's success on rush attempts versus the bottom 10 defenses--the 10 defensive units which allowed the most average rushing yards per attempt over the season.

-- Query 12: Eagles' average YardsGained per rush attempt against bottom 10 defenses (based on average rushing YardsGained allowed per game over season)

 INNER JOIN plays p2
  ON p2.DefensiveTeam = BottomDefRush.DefensiveTeam

ORDER BY AVG(p2.YardsGained) DESC;
Figure 12: Query 12 Results

Again, the results are unclear. Against the four worst run defenses (based on average YardsGained per rush attempt) the Eagles gain more yards than average against two (New Orleans Saints and New York Giants) and fewer yards than average against the other two (Washington Redskins and Buffalo Bills).

Next I extract the difference in average YardsGained by the Eagles against a particular team, based on RunLocation, versus the average YardsGained against those same defenses by all other teams.

-- Query 13: Opposing defense effectiveness based on RunLocation versus the Eagles' average YardsGained in that direction
SELECT  p2.PlayType,
        p2.RunLocation,
        AVG(p2.YardsGained) AS AllTeams_RushAvg,
        p3.PHI_RushAvg AS PHI_RushAvg, p3.PHI_RushAttempts
FROM (
    SELECT  DISTINCT DefensiveTeam AS DefTeam,
            posteam 
    FROM plays
    WHERE posteam = 'PHI') AS p1
JOIN plays p2
ON p2.DefensiveTeam = p1.DefTeam
JOIN (
    SELECT  posteam,
            PlayType,
            RunLocation AS PhiRunLocation,
            AVG(YardsGained) AS PHI_RushAvg,
            COUNT(*) AS PHI_RushAttempts
    FROM plays
    WHERE posteam = 'PHI' AND PlayType = 'Run' AND RunLocation IS NOT NULL
    GROUP BY RunLocation) AS p3
 ON p2.RunLocation = p3.PhiRunLocation
WHERE p2.PlayType = 'Run' AND p2.RunLocation IS NOT NULL
GROUP BY p2.RunLocation
ORDER BY AVG(p2.YardsGained) DESC;
Figure 13: Query 13 Results

The Eagles performed above average when rushing toward the middle against teams who were strong against middle runs. They were below average when rushing toward the left and right against teams who were strong against runs in those directions. Interestingly enough, the Eagles had the fewest number of rush attempts toward the middle (121 attempts) even though they beat the average in that direction.

I run a similar query for completed passes, comparing the Eagles' average YardsGained on Short and Deep passes to the average for all teams.

-- Query 14: Opposing defense effectiveness based on PassLength versus the Eagles' average YardsGained in that direction
SELECT  p2.PassOutcome,
        p2.PassLength,
        AVG(p2.YardsGained) AS AllTeams_PassAvg,
        p3.PHI_PassAvg AS PHI_PassAvg
FROM (
    SELECT  DISTINCT DefensiveTeam AS DefTeam,
            posteam
    FROM plays
    WHERE posteam = 'PHI') AS p1
JOIN plays p2
ON p2.DefensiveTeam = p1.DefTeam
JOIN (
    SELECT  posteam,
            PassOutcome,
            PassLength AS PhiPassLength,
            AVG(YardsGained) AS PHI_PassAvg,
            COUNT(*) AS PHI_CompletePasses
    FROM plays
    WHERE posteam = 'PHI' AND PassOutcome = 'Complete' AND PassLength IS NOT NULL
    GROUP BY PassLength) AS p3
 ON p2.PassLength = p3.PhiPassLength
WHERE p2.PassOutcome = 'Complete' AND p2.PassLength IS NOT NULL
GROUP BY p2.PassLength
ORDER BY AVG(p2.YardsGained) DESC;
Figure 14: Query 14 Results

The Eagles completed more Short than Deep passes, beat the average on Deep passes and performed below average for Short passes. A more insightful query would show compare their completion percentage for Short versus Deep passes versus all teams' against common defensive units.

def CompletionsChiSquare(obs, exp):
    obs_completions = len(obs.query("PassOutcome == 'Complete'"))
    obs_incompletes = len(obs.query("PassOutcome == 'Incomplete Pass'"))
    exp_completions = len(exp.query("PassOutcome == 'Complete'"))
    exp_incompletes = len(exp.query("PassOutcome == 'Incomplete Pass'"))

    exp_proportions = [1 / len(exp) * exp_completions, 1 / len(exp) * exp_incompletes]
    observed = [obs_completions, obs_incompletes]
    expected = [exp_proportions[0] * len(obs), exp_proportions[1] * len(obs)]
    return scipy.stats.chisquare(observed, expected)
