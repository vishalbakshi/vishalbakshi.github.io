SFTR Embed

Current MVP: https://sftrembed.firebaseapp.com/

This app currently:

  • Loads local (non-rapid) SFMTA MUNI transit data, whose isochrone is rendered upon pin placement
  • Renders rapid MUNI isochrones upon user selection (checkboxes) of rapid routes
  • Locks the isochrone maximum time to 30 minutes

There are, however, some key tasks that are still pending, which I have added to the Project:

  • Refactor the added code while writing tests
  • Allow user-defined route proposals to be uploaded into the map
