Project Journal: March 23, 2019

Over the past five weeks, a few things of note were accomplished:

I spruced up et130webapp with some prettier visual elements, and added MathJAX to the page to render prettier formulas. I am hoping to receive feedback from students once they return from their Spring Break in a little more than a week.

I finished the fourth freeCodeCamp project for the Information Quality and Assurance certification. The object of this project was to allow users submit one or two stock tickers (and "like" them) to receive the current stock price, total likes (for a single ticker) and relative likes (for two tickers). I used the addToSet operator when updating the database with likes because one of the requirements was to only allow 1 like per IP address. THe next step is to finish the fifth (and final) project to complete the certification.

I completed the zeroeth version of a file parser which reads an export file from PowerSchool and calculates ADA for each month of the school year. I call this the zeroeth version, since I wrote it without tests during a flight from Orlando, just to start wrapping my head around the functionality. The next step is to start from scratch with tests.

I (almooost) completed the first version of a SPA I created using React to brush up on my React and learn more about financial calculations in JS. I spent more time on the visual design which allowed me to wrap my head around the flow of props, and state changes, through different components. I wrote a basic unit test using Jest, and will dive into the world of Enzyme next.

Musically, I made some great progress over the last few weeks, inching closer to finding my natural vocal range. I post new versions of each song on my soundcloud page and the improvement between each version of a song is promising. I am learning how to incorporate elements of hip hop without enforcing the cadence I developed over the years. This is quite an exciting but energy-intensive journey.
