Project Journal: March 28, 2019

A short entry for today:

The following Google Codelabs are highly recommended if you are new to Firebase: Firebase Web Codelab and Cloud Functions for Firebase Codelab.

However, a more thorough (for beginners) tutorial is available at https://www.fullstackfirebase.com/ which, in my opinion, is the best tutorial I have come across yet. He's honest about the complexity of a non-trivial web app, and thorough with his explanations---often filling in the gaps in the (well written) Firebase documentation that beginners like me would not deduce on my own.

I've only read through this once, and have not yet coded along with it, but I am relieved to find the most understandable tutorial on testing Firebase functions: Test-Driven Cloud Functions

Lastly, a debug experience: while building a firebase app, I needed to retrieve a users' email field, but it's value was null even though I had signed up using Google Authentication. As it turns out, I was allowing "multiple accounts using the same email address" which was causing this issue. I switched that setting in "Sign-In Method" on the Firebase Console to "One account per email address", deleted my two dummy user accounts from the app, signed up again and voila the email addresses were stored. I could then retrieve a user's email field and was able to verify associated fields in other collections.
