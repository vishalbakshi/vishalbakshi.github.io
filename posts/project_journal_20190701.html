Form Tests

The project is lovely and simple: the user submits a form with their information (text, images), previews it, submits it (or edits it) and receives a confirmation message.

Right now I test that each component is rendered, and that the user can click "preview" and "submit" buttons.

Here are tests that I need:

Upon clicking the "preview" button, the form data displayed should match what was inputted by the user

The user can go back to the editable form and update the data

Upon submission, the data is prepared to be sent to the database

Upon submission, the user receives a confirmation message which includes the email sign-in link

Upon submission, the data is successfully sent to the database

Upon navigating to the email sign-in link, the user can view and update their form
