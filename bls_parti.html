ObjectiveÂ¶

The Bureau of Labor Statistics maintains and provides LABSTAT, a "central repository of data for the Bureau of Labor Statistics...considered the agency's database of record". One of the data sets in this database is the Consumer Price (CPI) Index Time Series, tracking the CPI in different geographic regions and categories of goods and services across the US. My goal for this post is to understand the data and mapping files, import the time series data, and create some basic visualizations of different trends over time.

What is CPI?Â¶

Consumer Price Index (CPI) is a measure of the costs of goods and/or services in a given year relative to the cost of the same quantity of goods and/or services at a past reference time.

$$CPI = \frac{\textit{cost of a good in a given year}}{\textit{cost of the good in the base year}} * 100$$

A series_id such as CUSR0000SA0 includes the survey abbreviation CU (CPI for all urban consumers), the seasonality code S if the value is adjusted for seasonality and U if it's unadjusted, the periodicity code R if Regular or S if Semi-annual, the area code 0000 (code for the particular geographic area this value comes from), and the item code SA0 (SA0 for all items in the index, SA0E for energy items, SAA for apparel, etc).

The period columns identifies either month M01 - M12, annual average M13 or semi-annual term S01 - S03. Finally, the value columns identifies the CPI for that particular record.

Each region starts and ends with the same year and period. I write a function to create a list of dates to use as the x-axis, then plot the CPI for each selected category in each region:

In [46]:
def MakeDates(df):
    starty = int(df.year[df.index.min()])
    startm = int(df.period[df.index.min()][-2:])
    n = len(df)
    start = datetime.date(starty, startm, 1)

    pdates = []

    for i in range(n):
        date = start + relativedelta(months=+i)
        pdates.append(pd.to_datetime(date))
    return pdates
