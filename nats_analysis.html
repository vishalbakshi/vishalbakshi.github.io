ObjectiveÂ¶

The National Adult Tabacco Survey (NATS) is performed by the CDC in order to gather information relevant to the habits and opinions of adults ages 18 and above in the United States. From the NATS website:

NATS was designed as a stratified, national, landline, and cell phone survey of non-institutionalized adults aged 18 years and older residing in the 50 states or D.C. It was developed to yield data representative and comparable at both national and state levels. The sample design also aims to provide national estimates for subgroups defined by gender, age, and race/ethnicity.

I want to perform some exploratory analysis for this survey, mainly to practice importing and cleaning data from SAS into Python, and also to explore any relationships between the different features. In this post, I focus on visualizing some of the data such as the number of cigarettes smoked per day by respondents who identified as current daily smokers, and also exploring the relationship between quit attempts and the current smoker status of the respondent.

I used the sas7bdat package located here to import the original dataset into Python. I've also used and the code used in Think Stats by Allen B. Downey, specifically to create histograms, CDF plots and hypothesis test classes, which can be obtained at the associated github repo. Code that I've either modified or written is located in the file [nats.py]

The records in this survey are not representative on their own, but include sampling weights in order to make national and state estimates. Using the probability weights associated with national estimate representation, WT_national, the sample mean and variance are calculated as follows:

  • Sample Weighted Mean:
$$\bar{x} = \frac{\sum_{i=1}^{N}w_ix_i}{\sum_{i=1}^{N}w_i}$$
  • Sample Weighted Variance:
$$s^2 = \frac{V_1}{V_1^2-V_2}\sum_{i=1}^{N}w_i(x_i-\bar{x})$$

Where:

$w_i$ = Sample weights

$V_1 = \sum{w_i}$

$V_2 = \sum{w_i^2}$

In Python:

In [5]:
def SampleMeanVarWeighted(xs, weights):
    xbar = sum(weights * xs)/sum(weights)
    
    v1 = sum(weights)
    v2 = sum(weights**2)
    dev = xs - xbar
    var = v1 / (v1**2 - v2)*sum(weights*dev**2)
    return xbar, var

Preliminary ThoughtsÂ¶

Based on the weighted summary statistics, the difference in quit attempts between Daily Smokers and Former Smokers is apparent but small and I don't know yet if this difference is statistically significant. The difference between Non-Daily and Former Smokers is larger, but again the statistical significance must be evaluated first in order to suggest any claims.

Hypothesis TestsÂ¶

The two hypotheses that I want to explore are:

  1. Is the difference in mean quit attempts between Daily and Former Smokers statistically significant?

  2. Is the difference in mean quit attempts between Non-Daily and Former Smokers statistically significant?

For both hypotheses, the null hypothesis is the same:

  • There is no difference in means between the two groups; they both come from the same distribution.

In nats.py I've used the code for a hypothesis test similar to the one written by Allen B. Downey in Think Stats which you can see here and here. The HypothesisTest class contains a method MakeModel which creates the null hypothesis, that the distribution of quit attempts between two groups comes from the same distribution, by combining the data from the two groups into one array using np.hstack. The RunModel method splits the combined distribution into two arrays using the lengths of the original groups, and sends that data to the TestStatistic method which calculates the (absolute value) difference in mean between the two groups. The PValue method runs this model for a default 1000 times, and counts how many iterations resulted in an absolute value difference in mean greater than the train data result. I wrote a DiffMeansTest class which takes the test set of dat and the difference in mean from the train set:

In [67]:
class DiffMeansTest(ts2.HypothesisTest):
    def __init__(self, data, train_stat):
        """Initializes.
        
        data: pandas Series from two different smoker groups
        """
        self.data = data
        self.MakeModel()
        self.actual = train_stat
        self.test_stats = None
        self.Cdf = None

    def MakeModel(self):
        """Builds a model of the null hypothesis."""
        group1, group2 = self.data
        self.n, self.m = len(group1), len(group2)
        self.pool = np.hstack((group1, group2))
    
    def RunModel(self):
        """Run a model of the null hypothesis.
        
        returns: simulated data (two groups)
        """
        np.random.shuffle(self.pool)
        data = self.pool[:self.n], self.pool[self.n:]
        return data
        
    def TestStatistic(self, data):
        """Computes the test statistic.

        data: two pandas Series
        """
        group1, group2 = data
        test_stat = abs(group1.mean() - group2.mean())
        return test_stat
    
    def PValue(self, iters=1000):
        """Computes the distribution of the test statistic and p-value.
        
        iters: number of iterations
        returns: float p-value
        """
        self.test_stats = [self.TestStatistic(self.RunModel()) for _ in range(iters)]
        self.test_cdf = ts2.Cdf(self.test_stats)
        
        count = sum(1 for x in self.test_stats if x>= self.actual)
        return count / iters

def ResampleRowsWeighted(df, column='WT_national'):
    """Resamples a DataFrame using probabilities proportional to given column.
    
    df: DataFrame
    column: string column name to use as weights

    returns: DataFrame
    """
    weights = df[column].copy()
    weights /= sum(weights)
    indices = np.random.choice(df.index, len(df), replace=True, p=weights)
    sample = df.loc[indices]
    return sample

def PlotSamplingDistribution(series, label=''):
    cdf = ts2.Cdf(series)
    ci = cdf.Value(0.05), cdf.Value(0.95)
    tp.Cdf(cdf, label=label)
    nats.VertLine(ci[0])
    nats.VertLine(ci[1])
    print('%s - 90%% CI: (%0.3f, %0.3f)' % (label, ci[0], ci[1]))

Future ExplorationsÂ¶

This analysis helped identified some interesting and possibly releavent relationships between quit attempts and smoker status, but further analysis is necessary (either with different models and/or with later cycles of the NATS) in order to give stronger evidence for any claims. I would also like to explore the relationships in future posts between smoker status and the following factors (NATS variable name shown in parenthesis):

  • Employement Status (EMPLOY2)
  • Number of jobs held (NOJOBS)
  • Household Income (INCOME2)
  • Race (RACEETHNIC)
  • Use of cessation meds such as sprays, lozenges, etc. (QTMED2)
  • Use of other forms of tobacco (chew, snus, cigars, etc) (Sltever2_r, currentslt_r, Snusever_r, Cigarever_r)
  • Marriage status (MARITAL2)
  • Healthcare coverage (HCCOVERAG)
